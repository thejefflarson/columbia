Installing GIS tools

Mac: There best way to install the tools we need is with a command line tool called homebrew. Essentially you open Terminal.app (in the Utilities folder in Applications) and follow the directions on Homebrew's Website. After you've installed homebrew, I recommend using osgeo's installer, so type this in your terminal: 

brew install matplotlib
sudo ln -s /System/Library/Frameworks/Python.framework/Versions/2.7/Extras/lib/python /Library/Python/2.7/site-packages/matplotlib-override
brew cask install qgis
brew install gdal

That will take a really long time to complete, and you may need to run it more than once. Another option is to use the installer on the QGIS website. You first need to install the GDAL Complete package from Kyngchaos

Windows: I've heard good things about OSGeo4W but I've never used it. You'll also need to install python. One other option is to run a linux machine in virtualbox.
